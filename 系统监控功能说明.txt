【功能说明】
在管理员后台底部添加了实时系统监控显示，包括：

✓ 程序内存占用 - 显示当前应用占用的内存
✓ 系统内存使用 - 显示服务器总内存使用率
✓ CPU使用率 - 显示CPU使用率和核心数
✓ 数据库连接 - 显示当前/最大数据库连接数

【特色功能】
• 实时更新（每3秒）
• 彩色进度条（绿色正常/黄色警告/红色危险）
• 响应式设计（移动端自适应）
• 鼠标悬停显示详细说明

【已完成的修改】
1. ✓ templates/includes/footer.html - 添加监控UI和实时更新脚本
2. ✓ mod/blueprint/admin.py - 添加 /admin/system-monitor API
3. ✓ static/css/admin/base.css - 调整底部栏样式
4. ✓ requirements.txt - 添加 psutil==5.9.8 依赖

【部署步骤】

1. 安装依赖（如果还没安装）
   Windows: pip install psutil==5.9.8
   Linux:   pip3 install psutil==5.9.8

2. 重启应用
   Windows: restart_app.bat
   Linux:   ./restart_app.sh

3. 登录管理员后台
   访问管理员后台任意页面，查看底部栏

【测试验证】
运行测试脚本验证功能：
   python test_system_monitor.py

【监控指标说明】

📊 程序内存
   - 显示当前Flask应用占用的内存大小
   - 格式：MB 或 GB
   - 进度条：相对于系统总内存的占用比例

📊 系统内存
   - 显示服务器总内存使用百分比
   - 格式：百分比
   - 进度条：系统内存使用率

📊 CPU使用率
   - 显示CPU使用百分比和核心数
   - 格式：百分比 (核心数)
   - 进度条：CPU使用率

📊 数据库连接
   - 显示当前连接数/最大连接数
   - 格式：当前/最大
   - 进度条：连接数使用率

【颜色说明】
🟢 绿色：使用率 < 70%（正常）
🟡 黄色：使用率 70% - 90%（警告）
🔴 红色：使用率 > 90%（危险）

【性能影响】
• CPU占用：< 0.1%
• 内存占用：< 5MB
• 网络流量：每次约 200 bytes
• 更新频率：3秒/次

对系统性能影响极小，可以放心使用。

【故障排除】

问题1：监控数据显示为 "--"
解决：
  1. 检查是否已安装 psutil 库
  2. 检查后端API是否正常响应
  3. 查看浏览器控制台错误信息

问题2：数据库连接数显示为 0/0
解决：
  1. 检查数据库连接是否正常
  2. 确认MySQL用户有查询权限

问题3：进度条不显示
解决：
  1. 清除浏览器缓存
  2. 强制刷新页面（Ctrl+F5）
